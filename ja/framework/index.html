Intent

今、何をしようとしていますか？

Token Values

いつものコーヒーを再注文しているのか、キャンセルされたフライトを慌てて再予約しているのか？同じアプリでも、あなたがしようとしていることに基づいて全く違って見えるべきです。インテントはセッション中に変化することもあります — ブラウジングから始めて、購入で終わる。

Physical State

手は使えますか？

Token Values
Both Hands FreeOne HandNo HandsGlovedWalkingDriving

ソファで両手が空いている？ボタンは小さくてOK。電車でつり革を片手で握っている？ボタンは大きくなる。運転中？音声のみ、タッチは一切なし。あなたの身体がインターフェースを決める。

ヒューマントークン
③

Cognitive Load

脳はどれくらい疲れていますか？

Token Values
FocusedBaselineDividedDepletedPanic

集中している？フルダッシュボードを表示。仕事後で疲弊？選択肢は2つだけ。パニック状態？ボタンは1つ。最善の答えだけ。あなたの精神エネルギーが画面に表示される量を決める。

ヒューマントークン
④

Social Exposure

誰が画面を見ていますか？

Token Values
PrivatePublicColleagueFamilyShared

一人？すべて表示。混雑した電車？価格をぼかし、通知を消音。家族と一緒？画面を分割 — 親には価格とコントロール、子どもには写真とシンプルなテキスト。

コントロールトークン
⑤

Priority Weight

今最も重要なことは？

Token Values
Reliability > CostFeasibility > VarietyPrivacy > SpeedSafety > Convenience

最安のフライトか、最も信頼できるものか？ニーズが衝突するとき、何かが勝たなければならない。デザイナーがデフォルトの順序を設定。あなたはいつでも優先順位をドラッグして並び替えられる。

コントロールトークン
⑥

Synchro Rate

AIにどれだけ任せますか？

Token Values
SuggestConfirmNotifyAutonomous

初日：すべて確認する。半年後：任せる。4つのポジションのダイヤル — 「提案だけ」から「自動で実行」まで。信頼は時間とともに構築され、ダイヤルもそれに合わせて動く。

Form Factor

周りにどんな画面がありますか？

Token Values
Phone OnlyPhone + TVPhone + CarPhone + WatchLock ScreenAmbient

スマホだけ？フルUI。近くにTV？スマホがリモコンに。車の中？音声のみ。スマホは周りにある他の画面に応じて形を変える。

Token Values

家まで28分。冷蔵庫にはチキンとライス。スーパーは遠すぎる。不可能な選択肢は表示されない。今、手持ちのもので実際にできることだけ。
